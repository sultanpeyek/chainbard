# Brand

The mark is a **bard's plumed cap with a gold feather/quill** (`PlumeMark`), on
warm near-black ground — "the bard sets a quill to your chain." The wordmark
sets "chain" in bone and "bard" in amber.

### Palette

| Token | Hex | Role |
| --- | --- | --- |
| ink | `#0b0a09` | page ground (warm near-black) |
| ink-line | `#2a2622` | hairline / borders |
| bone | `#ece4d6` | primary text |
| amber | `#e8a13a` | single accent — the feather |
| wallet | `#c9a24a` | kind hue — wallet |
| tx | `#7ba2c4` | kind hue — transaction |
| nft | `#b07cc6` | kind hue — NFT |
| token | `#5aa087` | kind hue — token |

### Typography

- **Fraunces** — display serif, for headings and the story narrative.
- **IBM Plex Mono** — data/mono, for identifiers, addresses, and receipts.

Regenerate all brand assets with `bun run gen:brand`.

### Usage

- **DO** keep the cap on the ink ground (`#0b0a09`).
- **DON'T** place the mark on a busy or light background.
- **DON'T** recolor the mark — amber is the only accent.
